# 🚀 AI Interview Platform

An AI-powered interview practice platform built with **Next.js**, allowing users to simulate real interview experiences with voice input, AI-generated questions, and performance tracking.

---

## ✨ Features

- 🤖 AI-generated interview questions
- 🎤 Voice-based answering system
- ✍️ Manual text answer input
- ⏱️ Timed interview sessions
- 📊 Dynamic scoring system
- 🎯 Progress tracking per question
- 🎉 Completion screen with results
- 🔄 Restart interview option
- 📱 Responsive modern UI

---

## 🛠️ Tech Stack

- Next.js (App Router)
- React.js
- TypeScript
- Tailwind CSS
- Lucide Icons
- (Optional) OpenAI API for AI evaluation
